# shio

The portfolio site of shio, creator for creators.

## Setup

### git clone

```powershell
git clone "https://github.com/dino3616/shio"
```

### launch conatiner

```powershell
docker compose -f "./docker/docker-compose.development.yml" -p "shio" up -d
```

### set env variables

See `.env.example` for more details.
